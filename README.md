# AstroInsight

AI-powered astrology readings and birth chart analysis.

## About

AstroInsight generates personalized astrological readings and relationship compatibility assessments. Enter your birth data to receive detailed chart analysis powered by modern AI.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Vercel deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.

## Deployment

Deployed on Vercel. Push to `main` branch to trigger automatic deployment.
